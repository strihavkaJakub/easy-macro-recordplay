By default, the macro playback has the same start delay as the time between pressing the Record key and the first recorded keystroke. This check box can be used to switch off the start delay. This also affects the playback delay. Without ignoring the start delay, the next replay will start after Delay Between Replays + the initial delay between pressing the record button and the first recorded keystroke.

This setting allows you to add an offset to the delay between each keystroke. This can be useful if your playback keystrokes are too fast or too slow compared to the recorded speed.

A positive value increases the delay between replays, a negative value decreases it.
